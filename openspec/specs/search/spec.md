## Requirements
### Requirement: Client-Side Filtering
The system SHALL provide a mechanism to filter the list of posts by title or tags.

#### Scenario: Filtering by keyword
- **WHEN** the user types a keyword into the filter input
- **THEN** the system updates the post list to show only posts where the title or tags contain the keyword (case-insensitive)
